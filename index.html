2. smartadd: Stage All Except Specific Files

This smartadd alias allows you to quickly stage all changes in your repository while automatically excluding specific files.

How to add this alias:

run the following command in terminal

git config alias.smartadd '!git add . && git reset .vscode/settings.json apps/learner/src/app/app.tsx libs/utils/src/lib/config/config.ts yarn.lock'

How it works:

  • First, it runs git add . to stage all changes in your working directory.
  • Immediately after, it uses git reset to unstage the specified files: .vscode/settings.json, apps/learner/src/app/app.tsx, libs/utils/src/lib/config/config.ts, and yarn.lock.

Example Usage:

# Stage all changes except the specified configuration and lock files
git smartadd

3. smartcommit: Intelligent Commits with Ticket IDs

This alias, smartcommit, streamlines the commit process by automatically including a ticket ID (if found in the branch name) in your commit message and excluding specific files from the commit.

How to add this alias:

run the following command in terminal

git config alias.smartcommit '!f() { \
  git add . && \
  git reset .vscode/settings.json apps/learner/src/app/app.tsx libs/utils/src/lib/config/config.ts yarn.lock && \
  ticket=$(git rev-parse --abbrev-ref HEAD | grep -oE "[A-Z]+-[0-9]+") && \
  read -p "Commit message: " msg && \
  git commit -m "$ticket - $msg"; \
}; f'

How it works:

  • It first runs git add . to stage all changes.
  • Then, it unstages the files: .vscode/settings.json, apps/learner/src/app/app.tsx, libs/utils/src/lib/config/config.ts, and yarn.lock (just like smartadd).
  • It attempts to extract a ticket ID from your current branch name using a regular expression ([A-Z]+-[0-9]+). This assumes your branch names follow a pattern like IRA-123 or GAP-456.
  • Finally, it creates a commit with a message that starts with the extracted ticket ID (if found) followed by the commit message you provide.

Example Usage:

# Assuming your current branch is 'feature/IRA-789-test'
git smartcommit "Implement user authentication"
# This will create a commit message like: "IRA-789 - Implement user authentication"

# If no ticket ID is found in the branch name
git smartcommit "Fix minor UI issues"
# This will create a commit message like: "Fix minor UI issues"
